## Pricing Models

### **Enterprise SaaS (Most Common)**
- **Small Business:** $10K-$50K/year
- **Mid-Market:** $50K-$200K/year
- **Enterprise:** $200K-$1M+/year
- **Pricing Factors:**
  - Number of facilities
  - Data volume
  - User seats
  - API calls
  - Support level

### **Usage-Based**
- **Per Facility:** $5K-$50K/year per data center
- **Per Metric Ton CO2:** $0.50-$5 per ton tracked
- **Per Employee:** $50-$200/employee/year

### **Freemium**
- **Free Tier:** Basic tracking (1-2 facilities)
- **Pro:** $500-$5K/month
- **Enterprise:** Custom pricing
